About This Map

The Great Lakes CRM Firm Landscape is an interactive directory of 159 cultural resource management (CRM) consulting firms operating across the Great Lakes region, compiled from four state SHPO consultant lists and supplementary web research.

Data sources: Ohio History Connection SHPO, Michigan SHPO, Wisconsin Historical Society, Pennsylvania SHPO. Data current as of June 2026.

Methodology: Firm locations represent HQ city centroids. Firms appearing on multiple state lists are deduplicated. Specialty designations reflect qualifications submitted to SHPO offices.

Built with: Leaflet.js · OpenStreetMap · Leaflet.MarkerCluster
